## 🚀 사용 방법

### 1. 기본 사용 (이미지 오버레이)

현재 상태 그대로 사용 가능:
- `public/maps/luxterra-preview.jpg` (3MB) 사용
- 전체 맵 크기: 16320 x 9258 픽셀
- 줌 레벨: -4 ~ 2

### 2. 타일 렌더링 사용 (대용량 이미지)

#### 단계 1: 원본 이미지 준비
```bash
# 원본 이미지를 다음 경로에 복사
cp /path/to/original.jpg /home/nihilncunia/coding/app/fa-tools/public/maps/luxterra-original.jpg
```

#### 단계 2: 타일 생성
```bash
cd /home/nihilncunia/coding/app/fa-tools
pnpm tsx scripts/generate-tiles.ts
```

#### 단계 3: 자동 적용
- 타일이 생성되면 자동으로 타일 렌더링 사용
- 수동 코드 변경 불필요
